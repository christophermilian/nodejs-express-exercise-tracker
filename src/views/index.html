Exercise tracker

Create a New User

POST /api/users

Get User Logs

GET /api/users/:_id/logs

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number


Created with Node.js, JavaScript, Express, and MongoDB.
